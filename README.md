# Battleship vs AI

A public, browser-playable Battleship game against an AI opponent. Built with React, TypeScript, Vite, and Tailwind CSS.

## Play the game

- Live site: https://bishwap.github.io/battleship/
- GitHub repo: https://github.com/bishwap/battleship

## How to play

- Your ships are placed automatically on the left board.
- Click the enemy board on the right to fire.
- Red = hit, blue/white = miss, dark red = sunk.
- Sink all five enemy ships before Admiral Intelligence (AI) sinks yours.

## Features

- AI with random and hunt/target firing strategy.
- Pixel-art ships and animated water cells.
- Commander chat with pixel avatars and shake effects on hits/exclamations.
- Responsive layout for desktop and mobile.
- Vitest unit tests for game logic and AI.
